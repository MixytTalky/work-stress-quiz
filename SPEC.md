## 功能規劃

### 功能 1：答題流程
- **輸入**：使用者點選每題的選項（每題 4 個選項，對應不同壓力分數）
- **處理**：累加分數，每題選項分配 0 / 1 / 2 / 3 分，共 10 題滿分 30 分，換算為百分比
- **輸出**：進度條顯示第 N/10 題，選完自動跳下一題
- **邊界條件**：不可跳題、不可回頭修改（簡化流程）

### 功能 2：結果計算與顯示
- **輸入**：答題完成後的總分
- **處理**：總分 / 30 × 100 = 壓力百分比 → 對應 7 級類型
- **輸出**：
  - 壓力百分比（大數字動畫）
  - 壓力等級名稱 + 詼諧稱號
  - 類型描述（2-3 句輕鬆文案）
  - 專屬形象圖
  - 壓力計量表動畫（指針從 0 轉到結果值）
- **邊界條件**：0% 和 100% 邊界值正確處理

### 功能 3：社群分享
- **輸入**：使用者點擊分享按鈕
- **處理**：產生分享文字 + OG Image URL
- **輸出**：複製連結 / Facebook / Line / Twitter 分享
- **邊界條件**：分享 URL 含 UTM 參數

### 功能 4：重新測驗
- **輸入**：點擊「再測一次」按鈕
- **處理**：分數歸零，流程重置
- **輸出**：回到第一題

## 測驗題目設計

> 風格：輕鬆詼諧、小品感、讓人會心一笑

## 結果分級

| 區間 | 壓力% | 稱號 | 描述 |
|------|-------|------|------|
| 0~15% | 0-15 | 佛系無壓・職場仙人 | 你根本是來公司度假的吧？上班對你來說就像去公園散步，羨煞旁人。拜託分享一下你的快樂秘訣！ |
| 15~30% | 16-30 | 微風輕拂・淡定打工人 | 偶爾有點小煩躁，但整體來說你handle得很好。適當的壓力反而是你的動力，繼續保持這個節奏！ |
| 30~45% | 31-45 | 小有負擔・努力搬磚者 | 壓力開始有點份量了，但你還撐得住。記得偶爾放自己一馬，下班後的時間是你的，別客氣。 |
| 45~60% | 46-60 | 壓力臨界・瀕臨當機 | ⚠️ 警告：壓力水位已達警戒線！你可能需要一個假期，或至少一杯好咖啡。記住：你的價值不等於你的產出。 |
| 60~75% | 61-75 | 明顯過載・身心俱疲 | 紅燈亮起！你的身心正在發出求救訊號。是時候認真思考一下工作與生活的平衡了，你值得被好好對待。 |
| 75~90% | 76-90 | 瀕臨爆炸・壓力鍋模式 | 你簡直是行走的壓力鍋，隨時可能噴氣！強烈建議跟信任的人聊聊，或考慮尋求專業協助。你的健康比任何 KPI 都重要。 |
| 90~100% | 91-100 | 壓力爆表・需要擁抱 | 抱抱你 🫂（這是唯一允許的 emoji）。你承受的壓力已經超出正常範圍了。請認真考慮跟主管談談工作量，或諮詢專業心理師。記住：請求幫助是勇敢的表現。 |

## 技術備註

- 計分方式：10 題 × 0~3 分 = 0~30 分 → 換算 0~100%
- 百分比公式：`Math.round(totalScore / 30 * 100)`
- 結果頁壓力計量表用 CSS 動畫呈現
- 題目順序固定不隨機
- 零 emoji 政策：所有視覺元素使用 AI 生成圖示
- 純前端實作，無需後端
